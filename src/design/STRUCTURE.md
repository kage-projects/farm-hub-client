## âœ¨ Key Concepts

### 1. Color Shades (50-900)
- 50-200: Very light (backgrounds)
- 300-400: Medium light (highlights)
- 500: Base color
- 600-700: Medium dark (buttons, text)
- 800-900: Very dark (text on light)

### 2. Semantic Tokens
- Abstract colors for specific purposes
- Auto-switch between light/dark
- Makes code more maintainable

### 3. Component Recipes
- Pre-defined component styles
- Variants for different use cases
- WCAG AA compliance built-in

### 4. ColorScheme
- High-level color selection
- Maps to specific shades
- Consistent across components
